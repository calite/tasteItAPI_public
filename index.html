TasteIt API - documentación

Endpoints:

GET /public/recipes/all

Obtiene todas las recetas disponibles en la API.

No acepta parámetros adicionales.

Más información

GET /public/recipes/all/{skipper:int}

Obtiene todas las recetas disponibles en la API con paginación.

Acepta el parámetro opcional 'skipper' para indicar el número de recetas a saltar.

Más información

GET /public/recipes/{id:int}

Obtiene una receta específica por su ID.

Acepta el parámetro obligatorio 'id' para indicar el ID de la receta.

Más información

GET /public/recipes/random/{limit}

Obtiene recetas aleatorias de la API.

Acepta el parámetro obligatorio 'limit' para indicar el número máximo de recetas a obtener.

Más información

GET /public/recipes/byname/{name}/{skipper}

Busca recetas por nombre.

Acepta el parámetro obligatorio 'name' para seguir buscando por nombre y el parámetro opcional 'skipper' para paginar los resultados.

Más información

GET /public/recipes/bycountry/{country}/{skipper}

Busca recetas por país.

Acepta el parámetro obligatorio 'country' para especificar el país y el parámetro opcional 'skipper' para paginar los resultados.

Más información

GET /public/recipes/byingredients/{ingredients}/{skipper}

Busca recetas por ingredientes.

Acepta el parámetro obligatorio 'ingredients' para especificar los ingredientes separados por comas y el parámetro opcional 'skipper' para paginar los resultados.

Más información

GET /public/recipes/bytags/{tags}/{skipper}

Busca recetas por etiquetas.

Acepta el parámetro obligatorio 'tags' para especificar las etiquetas separadas por comas y el parámetro opcional 'skipper' para paginar los resultados.
